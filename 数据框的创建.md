```python
arr1 = np.array(np.arange(12)).reshape(4,3)
arr1````````
type(arr1)

df1

```
2. 通过字典方式创建数据框
```python
dic1 = {'a':[1,2,3,4],'b':[5,6,7,8],
        'c':[9,10,11,12],'d':[13,14,15,16]}
dic1

df2

3. 通过数据框方式创建数据框
```python

df4

s1
type(s1)
```

### 创建序列
1. pandas中有两类非常重要的类
> Series: 一维数组, 数据框中的某列

> DataFrame: 数据框

2. 通过一维数组创建序列
```python
import numpy as np, pandas as pd
arr1 = np.arange(10)
arr1
type(arr1)
s2 = pd.Series(arr1)
s2
type(s2)
```
3. 通过字典方式创建序列
```python
dic1 = {'a':10,'b':20,'c':30,'d':40,'e':50}
dic1
type(dic1)
s3 = pd.Series(dic1)
s3
type(s3)
```

4. 直接创建
```python
x = pd.Series([1,2,3], index=['one', 'two', 'three'])
```

### 引用数据框
1. 引用某些列
```python
dic2 = {'a':[1,2,3,4],'b':[5,6,7,8],
        'c':[9,10,11,12],'d':[13,14,15,16]}
df2 = pd.DataFrame(dic2)
df2.columns = ['aaa','bbb','ccc','ddd'] ## 重命名列名

# 第一种引用方式
df2[['aaa','ddd']] ## 双层[[]]保证取出来的数据依然是数据框

# 第二种引用方式
df2.ix[:,['aaa','ddd']]

# 第三种引用方式
df2.loc[:,['aaa','ddd']]
```
2. 引用某些行
```python
dic2 = {'a':[1,2,3,4],'b':[5,6,7,8],
        'c':[9,10,11,12],'d':[13,14,15,16]}
df2 = pd.DataFrame(dic2)
df2.index = ['aaa','bbb','ccc','ddd'] ## 重命名行名

df2.ix[['aaa','ddd']] ## 双层[[]]保证取出来的数据依然是数据框
df2.loc[['aaa','ddd']] ## 双层[[]]保证取出来的数据依然数据框
```
3. 引用某些行和某些列
```python
dic2 = {'a':[1,2,3,4],'b':[5,6,7,8],
        'c':[9,10,11,12],'d':[13,14,15,16]}
df2 = pd.DataFrame(dic2)

df2.ix[[0,3],['a','d']] 
type(df2.ix[[0],['a']])  ##即使只取出一个数也还是数据框
```
